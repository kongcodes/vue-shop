# vue-shop

### 说明
- 项目参考《vue.js前端开发实战》
- 只是一个demo
- mockjs模拟数据

### 预览
手机端打开或谷歌手机模式 [进入](https://helloklh.cn/vshop)

### 技术栈
- vue
- Vue Router
- Vuex
- mint-ui
- mui
- axios
- mockjs
